# 🃏 Four Card Feature Section - ყველაზე მაგარი საიტი

ეს არის თანამედროვე და სუფთა დიზაინის მქონე სექცია, რომელიც წარმოადგენს პროექტის ძირითად მახასიათებლებს. საიტი აგებულია ვიზუალური იერარქიის პრინციპით, სადაც თითოეულ ფუნქციას თავისი ადგილი და ფერი აქვს.

🌐 **Live Demo:** [venerable-fenglisu-d01c0d.netlify.app](https://venerable-fenglisu-d01c0d.netlify.app/)

## ✨ ძირითადი მახასიათებლები (Features)

საიტი დაყოფილია ოთხ მთავარ ბლოკად, რომლებიც სხვადასხვა ხელსაწყოს აღწერს:
- **Supervisor:** პროექტის მონიტორინგი და დაბრკოლებების დადგენა.
- **Team Builder:** საჭირო რესურსებისა და გუნდის წევრების ძებნა.
- **Calculator:** გამოთვლების წარმოება საჭიროებისამებრ.
- **Karma:** პროექტის წარმატებისა და პოზიტიური რეიტინგის მაჩვენებელი.

## 🛠️ გამოყენებული ტექნოლოგიები

- **HTML5:** სემანტიკური და სუფთა სტრუქტურის ასაგებად.
- **CSS3 (Grid/Flexbox):** ბარათების უნიკალური განლაგებისა და დიზაინისთვის.
- **Responsive Design:** საიტი სრულად მორგებულია მობილურ და დესკტოპ ეკრანებზე.
- **Netlify:** პროექტის სწრაფი ჰოსტინგისთვის.

## 🎨 დიზაინის სტილი

პროექტი იყენებს:
- მკვეთრ ფერებს ბარათების ზედა ზოლისთვის (წითელი, ლურჯი, ყვითელი, ცისფერი).
- ჩრდილებს (Box Shadows) სიღრმის ეფექტისთვის.
- "Gatsby" სტილის სუფთა ტიპოგრაფიას.
